Privacy Policy for the Luvi Helper Bot

Last updated: March 2026

This Privacy Policy explains how the Luvi Helper Bot (“the Bot”) handles data when used on Discord servers.

1. Information the Bot Collects
The Bot only processes the minimum information required to function:

Discord user IDs

Discord server IDs

Role IDs

Message IDs (for features like autorole panels)

Interaction data (button presses, slash commands)

The Bot does not collect or store:

Messages or message content

Personal information outside Discord

IP addresses

Email addresses

Any sensitive personal data

2. How Data Is Used
The Bot uses data exclusively to:

Assign or remove roles

Respond to slash commands

Provide server utilities (notifications, stats, world attack features)

Maintain bot functionality and prevent abuse

No data is sold, shared, or transferred to third parties.

3. Data Storage
The Bot stores only:

Configuration settings

Role IDs

Message IDs

Server preferences

All data is stored securely and used only for the Bot’s features.

4. Data Retention
Data is retained only as long as the Bot is active in a server.
If the Bot is removed from a server, all related data is automatically deleted.

5. User Rights
Users may request:

Removal of their stored data

Clarification of what data is stored

Server administrators may reset all bot data at any time
